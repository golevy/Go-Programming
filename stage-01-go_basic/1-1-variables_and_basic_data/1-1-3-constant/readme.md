## 字面常量

所谓字面常量（literal），是指程序中硬编码的常量。

- 123       // 整型类型的常量
- 156.78    // 浮点类型的常量
- true      // 布尔类型的常量
- "abc"     // 字符串类型的常量

## 常量命名规范

- 只能又数字，字母，_（下划线）组成
- 不能以数字开头
- 大小写字母是不同的，建议全部大写
- 不能是关键字
- 见名知意
- 多个单词组成的名称，使用全大写字母书写，中间使用 _（下划线）分隔，例如：
    ```go
    const APP_VERSION = "0.7.0.1110 Beta"
    ```
